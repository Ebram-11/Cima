# 🎬 Cima — Cinema Booking System

**Cima** is a modern cinema booking platform that helps users discover, browse, and book movies across different cinemas — all in one place.

---

## ✨ Features

- 🎥 **Browse Movies** — Explore now-showing and upcoming movies with details, trailers, and ratings
- 🏢 **Multi-Cinema Support** — View showtimes across multiple cinema locations
- 💺 **Seat Selection** — Interactive seat maps for choosing your perfect spot
- 🎟️ **Easy Booking** — Seamless ticket booking and checkout experience
- 👤 **User Accounts** — Register, log in, and manage your bookings
- 📱 **Responsive Design** — Works beautifully on desktop, tablet, and mobile
